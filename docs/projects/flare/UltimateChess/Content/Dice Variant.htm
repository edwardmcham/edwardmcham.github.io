Dice Variant

Use dice to resolve captures when playing cards aren't available.

Standard Dice

Each player rolls one six-sided die per capture attempt. The higher roll wins. Reroll ties.

  • Rolling doubles on two dice counts as a Sniper win. Apply Sniper Rules.

Custom Dice (D&D Dice)

Players may agree on custom battle rules using polyhedral dice (D4, D6, D8, D10, D12, D20). Establish house rules before the game begins:

  • Which die or dice each player rolls
  • What result counts as a Sniper equivalent
  • How to resolve ties

Note: Write down custom dice rules and agree on them before the game. Apply the same rules equally to both players.
