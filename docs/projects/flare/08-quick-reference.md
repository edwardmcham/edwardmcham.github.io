# Quick Reference

| Situation | Resolution |
|---|---|
| Standard capture attempt | Play 3 face-down + 1 face-up. Higher card wins. |
| Attacker wins | Capture succeeds. |
| Defender wins | Attacker's piece is removed. Defender's piece stays. |
| Tie (no Jokers) | No pieces move or are removed. Turn consumed. |
| Attacker wins with Joker | Capture succeeds. Attacker takes one bonus non-King piece. |
| Tie (both Jokers) | Armageddon: both pieces removed. Each player removes 3 more non-King pieces. |
| Attacking a King | Best of three card battles. First to two wins captures the King. |
| King attacks, loses | King is captured. Attacking player loses immediately. |
| No cards available | Use the [Dice Variant](07-dice-variant.md). |
